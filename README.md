# QGROUP — каталог оборудования

Веб-приложение: публичный каталог оборудования + кабинет партнёра + админ-панель +
конструктор коммерческих предложений (КП). Двуязычный интерфейс (RU/RO).

> Каталог-справочник: без складских остатков, заказов и оплаты.

---

## Техническая часть

### Стек
Node.js + **Express**, шаблоны **EJS** (рендер на сервере), база **SQLite** (better-sqlite3),
изображения — **WebP** (sharp), импорт прайса из **Excel** (exceljs). Вся работа с БД
изолирована в `src/config/db.js` и `src/models/` — переход на PostgreSQL/MySQL без
переписывания контроллеров.

### Роли и доступ
| Роль | Розн. цена | Партн. цена | Каталог (ред.) | КП |
|---|---|---|---|---|
| Посетитель | ✅ | — | — | — |
| Партнёр | ✅ | ✅ | — | партнёрский |
| Продавец | ✅ | ✅ | — | общий (+ LED) |
| Менеджер | ✅ | ✅ | ✅ | — |
| Админ | ✅ | ✅ | ✅ | все |

Партнёрская цена (`price_partner`) вырезается на сервере в единой точке `serializeProduct()`
([src/models/product.js](src/models/product.js)) — в публичный ответ не попадает.

### Возможности
- **Каталог:** категории, поиск, сортировка (название / цена / новинки), карточки с фото.
- **Корзина-подборка** → перенос выбранных товаров в конструктор КП.
- **Конструктор КП** (общий / LED / партнёрский, десктоп + мобайл): каталог из товаров,
  карточки менеджеров, фото товаров в КП, печать в PDF средствами браузера.
- **Админка:** товары (CRUD + фото), пользователи, контактные лица (публичная страница
  «Контакты»), лента активности (аудит, включая действия по КП), экспорт товаров в Excel.
- **Импорт прайса** из Excel со встроенными фото; машинный перевод карточек на RO.
- **i18n** RU/RO, фирменный стиль QGROUP.

### Структура
```
src/    config/(db, i18n)  models/  controllers/  middleware/  routes/  utils/  index.js
views/  EJS-шаблоны (витрина + админка + partials)
public/ css  js  images/{products, contacts}
kp-builders/  HTML конструкторов КП
scripts/      migrate, seed, import-price, translate
data/   qgroup.db (SQLite, вне git)
```

### Запуск (локально)
Node 20+ (готовые бинарники, компилятор не нужен).
```bash
npm install
cp .env.example .env        # настроить SESSION_SECRET, ADMIN_*
npm run migrate && npm run seed
npm run dev                 # http://localhost:3000
```
Логин/пароль админа задаются в `.env` (`ADMIN_LOGIN` / `ADMIN_PASSWORD`) — смените после
первого входа.

### Импорт прайса
```bash
npm run import-price -- data/price.xlsx     # идемпотентно по коду товара
```

### Деплой
Node-процесс под **PM2** + **nginx** (reverse-proxy, статика напрямую) + **HTTPS**
(Let's Encrypt). Приложение использует `trust proxy` и secure-cookie в авто-режиме.

### Безопасность
bcrypt-пароли, httpOnly / SameSite-сессии, rate-limit на `/login`, экранирование вывода
(XSS), мягкое удаление товаров (`is_active = 0`).
